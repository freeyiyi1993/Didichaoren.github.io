Wiki

多浏览器支持

      支持各大主流浏览器，目前的版本支持Firefox，chrome，IE(Version10+),Safari。不断争取完美。

响应式

      Web响应式设计，支持PC端多种屏幕分辨率和pad端多种屏幕分辨率，手机部分适配ing。

自己的风格

      简洁，清晰的样式，外观上拥有自己独立的风格，设计有待修改。争取经过版本迭代有PK设计师的资本~ (｡◝‿◜｡)

易于使用

      使用舒适化的类名与标签搭配，支持部分移动端手势。想要真正方便的提供给人使用绝非易事，但是我们会不断努力。

SNX是完全开源的。它的代码托管、开发、维护都依赖 GitHub 平台。
